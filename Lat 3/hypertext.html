HyperText Markup Language (HTML)TM

paragraf

HTML atau HyperText Markup Language merupakan bahasa yang digunakan dalam membuat antarmuka halaman pada web. Aplikasi web menggunakan antarmuka yang dibuat dalam HTML dan bahasa pemrograman untuk menontrol dan mengolah data. Oleh sebab itu, mengetahui HTML itu diperlukan dalam membangun aplikasi web dan ada baiknya untuk memiliki pemahaman tentang bahasa tersebut.

HTML sebagai bahasa penanda
yang digunakan untuk membuat
halaman web dengan tag atau simbol
sebagai tanda dari struktur halaman web.

HTML adalah bagian penting dari struktur web, termasuk menentukan tata letak dan tampilan konten. Bila dianalogikan HTML seperti batu bata dalam pembuatan rumah. Kemudian batu bata tersebut tersusun dengan rapih hingga membentuk fondasi β dasar.
